## ⚡ The Architectural Pillars of the Portfolio

### 1. The Dynamic Sensory System (UI & Motion)
The front-end is engineered using **React.js**, **Tailwind CSS**, and **GSAP (GreenSock Animation Platform)** to create a fluid, tactile user experience:
*   **Active Mouse Parallax:** The hero element computes screen displacement relative to the user's cursor, causing typographic elements and ambient glow fields to shift dynamically.
*   **Cinematic Audio-Visual Intro:** Background video stream begins as a muted ambient loop and dynamically unmutes with the first user interaction, bridging the gap between static browsing and media experience.
*   **Glassmorphism Layouts:** Modern frosted-glass panels (`glass-panel`) separate information with high contrast and subtle, animated borders.

### 2. The Interactive AI Schematics Panel
Rather than listing terms, the **AI Systems Section** houses dynamic SVG-rendered models and real-time console telemetry logs:
*   **Perception-Action Loop Console:** Demonstrates how autonomous AI agents sense inputs, reason using LLMs, and execute tools.
*   **Multi-Agent Orchestration:** Visualizes message distribution and collaborative decision-making across sub-agents.
*   **RAG Query Routing:** Animates vector database searches, context grounding, and source-grounded LLM synthesis.

## 🛠️ The Featured Showcase Projects

The portfolio highlights two core systems built to solve real-world problems:

### 🛡️ Behavioral Authentication System for Web Security
*   **The Problem:** Static credentials are prone to theft. Traditional security cannot distinguish between a legitimate user and an impostor with stolen passwords.
*   **The Solution:** A passive, continuous authentication layer that analyzes **keystroke dynamics** and **mouse movement trajectories** to establish a biometric session signature.
*   **Stack:** Python, Flask, `scikit-learn` (anomaly detection models), SQLite, NumPy.
*   **Metric:** Zero-friction passive monitoring.

### 🏥 AI-Powered Medical Assistant Chatbot (RAG)
*   **The Problem:** Online medical data is unstructured, and generic LLMs frequently hallucinate critical health advice.
*   **The Solution:** A Retrieval-Augmented Generation pipeline that parses medical literature, encodes it into a vector space, and queries context to ground responses.
*   **Stack:** Python, LangChain, Vector Databases, HuggingFace/OpenAI Embeddings.
*   **Metric:** Verified, source-grounded medical answers with citation tracking.

## 🔧 Run & Develop Locally

Get the portfolio running on your local machine in under a minute:

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/gowriram/portfolio.git
    cd portfolio
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Launch the Dev Server:**
    ```bash
    npm run dev
    ```

4.  **Build for Production:**
    ```bash
    npm run build
    ```
